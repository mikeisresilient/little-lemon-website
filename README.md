# Little Lemon Restaurant Website

A Django-based restaurant website developed as part of the Meta Full Stack Developer Professional Certificate on Coursera.

## Overview

This project demonstrates the fundamentals of web development using Django, including URL routing, views, templates, models, database integration, and the Django administration panel.

The application allows users to browse restaurant menu items and provides administrators with the ability to manage content through Django's built-in admin interface.

## Features

* Django project structure and configuration
* Dynamic menu display using database models
* Template rendering with Django Template Language (DTL)
* URL routing and view handling
* Django Admin integration
* Responsive and user-friendly interface
* SQLite database integration

## Technologies Used

* Python
* Django
* HTML5
* CSS3
* SQLite
* Git & GitHub


## Installation

### Clone the Repository

git clone https://github.com/YOUR_USERNAME/YOUR_REPOSITORY.git
cd YOUR_REPOSITORY

### Create a Virtual Environment

python -m venv venv

### Activate the Virtual Environment

Windows:
venv\Scripts\activate

macOS/Linux:
source venv/bin/activate

### Install Dependencies

pip install -r requirements.txt

### Apply Migrations

python manage.py migrate

### Run the Development Server

python manage.py runserver

Open your browser and navigate to:

http://127.0.0.1:8000/
